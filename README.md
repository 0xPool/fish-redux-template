# fish-redux-template

## Usage

- `git clone git@github.com:huangjianke/fish-redux-template.git`
- `code fish-redux-template`
- `npm install` in the terminal, then `F5` to run the extension

## Install

search `fish-redux-template` in Visual Studio Marketplace 

Enjoy yourself~
